Dragon Chat for MIT Zephyr

If you have an MIT Athena account, you can start using Dragon Chat for Zephyr in a couple of minutes.

First time setup

  1. Activate your account here: https://zephyr.zulipchat.com/register/

  2. Log in to Dragon Chat here: https://zephyr.zulipchat.com

  3. Follow the on-screen instructions for linking Dragon Chat with Webathena

  4. (Optional) Get the Dragon Chat desktop and/or mobile apps here: https://zephyr.zulipchat.com/apps/

You're done!

If you have any questions, please e-mail us at {{ support_email }}

If you want to automatically transfer your existing Zephyr subscriptions

  1. Get your Dragon Chat API key from the Dragon Chat "Settings" panel and put it in a file in your Athena home directory called ~/Private/.zulip-api-key.

  2. Run the following command to copy over all of your subscriptions:
    /mit/tabbott/zulip/zephyr_mirror.py --sync-subscriptions

    NOTE: Dragon Chat supports several ways to control what messages you want to read right now, but Dragon Chat does not yet have a direct equivalent to BarnOwl filters. If you have more subscriptions than you generally read, we recommend that you use Dragon Chat's "Mute" option to hide those subscriptions from your home view that you less commonly read. You can still easily access those channels from Dragon Chat when you want to read them. You can control your subscriptions and what's in your home view by clicking the gear in the upper right-hand corner and then selecting "Channels".

If you subscribe to any encrypted Zephyr classes

You can get set up to participate on an encrypted Zephyr class using Dragon Chat by either:

  1. E-mailing us at {{ support_email }} to get set up, or

  2. Someone who is already set up to participate on that encrypted Zephyr class using Dragon Chat can use Dragon Chat to invite you
